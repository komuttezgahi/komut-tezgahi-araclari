# Guvenlik Notlari

Bu repo herkese acik oldugu icin asagidaki dosya ve bilgileri yuklemeyin.

## Yuklenmemesi Gerekenler

- Gercek YouTube Data API anahtari
- OAuth token dosyalari
- client_secret dosyalari
- Google hesabina ait credential dosyalari
- Kisisel dosya yollari, e-posta adresleri veya telefon numaralari
- Gercek kanal hesabina ait ozel cikti dosyalari
- Buyuk video, ses ve gorsel taslaklari

## Onerilen Kullanim

1. Repodaki config.example.json dosyasini kopyalayin.
2. Kendi bilgisayarinizda config.json olarak yeniden adlandirin.
3. Gercek API anahtarinizi sadece yerel config.json icine yazin.
4. config.json dosyasini repoya yuklemeyin.

## API Anahtari Yanlislikla Paylasilirsa

- Anahtari hemen Google Cloud Console uzerinden iptal edin veya yenileyin.
- GitHub gecmisinde kalmis olabilecegini unutmayin.
- Yeni anahtari sadece yerel config dosyanizda kullanin.

## PowerShell Notu

Internet'ten indirilen .ps1 ve .bat dosyalarini calistirmadan once okuyun. Bu repodaki araclar admin yetkisi gerektirmeyecek sekilde tasarlanmalidir.
