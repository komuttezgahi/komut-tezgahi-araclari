# Kanal Videolarini Excel'e Aktarma

Bir YouTube kanalindaki videolari yayin tarihine gore listeleyip Excel'e aktarmak icin hazirlanacak arac klasoru.

## Planlanan Cikti Alanlari

- published_turkey_date
- published_turkey_time
- published_turkey_full
- published_utc_full
- title
- video_id
- url
- duration_iso
- view_count
- like_count
- comment_count

## Durum

Kodlar ve kullanim adimlari ilgili video yayinlandiginda eklenecek.
