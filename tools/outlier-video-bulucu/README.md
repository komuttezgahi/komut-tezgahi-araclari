# Outlier Video Bulucu

YouTube Data API v3 ile kanal veya rakip video verilerini inceleyip kanal ortalamasinin uzerinde performans gosteren videolari bulmaya yonelik arac klasoru.

## Amac

- Rakip kanal video performanslarini incelemek
- Kanal ortalamasinin uzerindeki videolari yakalamak
- Yeni video fikirleri icin veri temelli adaylar cikarmak

## Durum

Kodlar ve kullanim adimlari ilgili video yayinlandiginda eklenecek.
